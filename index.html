Tim Berners-Lee 的偉大貢獻

  • 1955 - 出生於英國倫敦
  • 1976 - 畢業於牛津物理系
  • 1978 - 於 D.G. Dash 製作排版軟體
  • 1989 - 於 CERN 發明 HTTP
  • 1991 - 向全世界免費開放 World Wide Web
  • 1994 - 成立 W3C (World Wide Web Consortium)
  • 1999 - 提倡 語意網 (Semantic Web)
  • 2009 - 成立 WWWF (World Wide Web Foundation)
  • 2012 - 成立 ODI (Open Data Institute)
  • 2013 - 成立 A4AI (Alliance for Affordable Internet)
  • 2016 - 成立 Solid (Social Linked Data)
  • 2018 - 成立 初創公司 Inrupt 展示 Solid 功能
  • 2019 - 提倡 Contract for the Web
